# xAI

Mitchell-extended cards for xAI models.

| Model | Tier | Context | Knowledge cutoff |
| ----- | ---- | ------- | ---------------- |
| [grok-4]({% link xai/grok-4.md %}) | frontier reasoning + tool use | 256k (per consensus) | November 2024 |

xAI publishes safety-focused **Model Cards** at `data.x.ai` rather
than arXiv preprints, and their consumer-product system prompts at
[`xai-org/grok-prompts`](https://github.com/xai-org/grok-prompts) as
a transparency mechanism. Architecture and parameter count are not
officially disclosed.
