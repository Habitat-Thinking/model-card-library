# Mistral

Mitchell-extended cards for Mistral AI models.

| Model | Tier | Context | Knowledge cutoff |
| ----- | ---- | ------- | ---------------- |
| [mistral-large-3]({% link mistral/mistral-large-3.md %}) | open-weight MoE (41B active / 675B total) | 256k | Not publicly available |

Released as **open weights under Apache 2.0** in BF16, FP8, and
NVFP4 formats on HuggingFace. The model was released without an
accompanying arXiv technical report — provider docs and the
HuggingFace model card carry the technical claims.
